Charlotte "Lotte" Reiniger (2 June 1899 – 19 June 1981) was a German film director and the foremost pioneer of silhouette animation.

History

  • 2015 - Hernan Torrisi released a plugin named Bodymovin.
  • 2017 - Engineers at Airbnb created first libraries that could render these JSON animation files, which they called “Lottie”.
  • 2017 - LottieFiles launched.
  • 2018 - Microsoft and the .NET Foundation released a library for rendering Lottie on Windows.
  • 2019 - further support for Lottie was by Qt, Samsung, Python.
  • 2020 - dotLottie was created.

Advantages of Lottie

  • Small file size
  • Infinitely scalable
  • Multi-platform support and libraries
  • Interactivity
